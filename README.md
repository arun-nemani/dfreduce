# dfreduce

Description: Automatically reduce the size of your pandas dataframe

## Installation
pip install dfreduce

## Usage
```
import dfreduce
import pandas as pd

df = pd.read_csv("http://www.ats.ucla.edu/stat/data/binary.csv")
df_reduced = DFreduce(df)
```

## Uninstallation

```
pip uninstall dfreduce
```
